Practical understanding and exercises

Please use our DYLEN tool integration or open the tool in a new tab to complete the following tasks.

Basic word search

You are interested in the use of the word “AHS” in 2016 in the KRONE subcorpus. Please find out which other words are often used in this context and name three of them.

Visualisation

Please select the three words you just named and visualize their metrics. Name the one with the highest Pagerank.

Graph Comparison

We now want to inspect a second word in the same subcorpus and year: “AHS-Lehrer”. Please visualize its network underneath the first one and select a word that has more than four connections to other words in the network.

Table View Visualisation

Please switch to the table view and select all words that appear in both networks. Out of these selected words, find the one with the highest clustering coefficient value. Name the word.
